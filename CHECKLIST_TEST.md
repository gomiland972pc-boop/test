## 0. Проверка WEBHOOK_URL

- [ ] Открыть Bot-host → настройки бота → env-переменные
- [ ] `WEBHOOK_URL` заканчивается на `/webhook`
      (например `https://testoviybotmax.bothost.tech/webhook`, а НЕ `...tech/`)

---

## 1. Старт бота (ошибки нет)

- [ ] Зайти в Bot-host → логи
- [ ] Убедиться, что в логах **нет** «`No such file or directory`»
- [ ] Убедиться, что в логах **нет** «`method HEAD is already registered`»
- [ ] Убедиться, что есть строка:
      `Бот авторизован: id=...`
- [ ] Убедиться, что есть строка:
      `Webhook зарегистрирован на https://...`
- [ ] Убедиться, что есть строка:
      `Бот запущен. admin_ids=[6228168]`
- [ ] Убедиться, что **нет** строки `POST / HTTP/1.1" 404`

---

## 2. Обычный пользователь — первый вход

- [ ] Открыть Max → написать боту `/start`
- [ ] Бот присылает документы (Согласие + Оферта)
- [ ] Нажать «Принять Согласие»
- [ ] Нажать «Принять Оферту»
- [ ] После обоих — открывается Главное меню:
      - Инструкции по боту
      - Профиль и каналы
      - Связаться с поддержкой

---

## 3. Создание тикета пользователем

- [ ] В главном меню нажать «Связаться с поддержкой»
- [ ] Нажать «Открыть новый тикет»
- [ ] Написать: *«Тест: у меня вопрос по боту»*
- [ ] Отправить
- [ ] Должно прийти сообщение: `✅ Тикет №... → *На рассмотрении*`
- [ ] Должна открыться история тикета. Убедиться, что:
      - [ ] Есть строка «Тикет №...»
      - [ ] Есть строка «От кого:» — твоё имя
      - [ ] Есть строка «Кому: Служба Поддержки»
      - [ ] Есть строка «Статус: На рассмотрении»
      - [ ] Есть строка «Создан: дата»
      - [ ] Есть строка «Обновлён: дата»
      - [ ] В истории: твоё имя: твоё сообщение

---

## 4. Пользователь — статус в истории

- [ ] Нажать «Ответить» у тикета
- [ ] Написать: *«Ещё одно сообщение»*, отправить
- [ ] Бот присылает: `✅ Тикет №... → *На рассмотрении*`
- [ ] В обновлённой истории **видна строка Статус**
- [ ] Статус показан: *На рассмотрении*

---

## 5. Админ — список пользователей

- [ ] Войти как админ (id=6228168), написать `/admin`
- [ ] Нажать «Пользователи»
- [ ] Убедиться:
      - [ ] Заголовок: «Последние пользователи»
      - [ ] Есть подсказка про 🟢 = активен 15 минут
      - [ ] Имя пользователя — **кликабельная ссылка** (синяя, нажимается)
      - [ ] Рядом есть `id <число>` серым текстом
      - [ ] Есть дата/время последнего захода
      - [ ] Если пользователь только что писал — 🟢 горит
      - [ ] Сортировка: недавно активные — сверху

---

## 6. Админ — просмотр тикета

- [ ] В админ-панели → «Активные» → выбрать тикет пользователя
- [ ] В шапке тикета:
      - [ ] «От кого:» — **имя пользователя кликабельной ссылкой**
      - [ ] «Кому:» — **Служба Поддержки** (жирным)
      - [ ] «ID:» — серым
      - [ ] «Статус: На рассмотрении»
- [ ] В истории:
      - [ ] Сообщения пользователя — подписаны **его именем** (ссылка, жирный)
      - [ ] Где ответ поддержки — подписано **Служба Поддержки** (жирный)

---

## 7. Админ меняет статус

- [ ] В открытом тикете нажать «Изучает специалист»
- [ ] Приходит подтверждение: `✅ Тикет #... -> Изучает специалист`
- [ ] **Проверить у пользователя:**
      - [ ] Открыть Max обычным пользователем
      - [ ] Найти тот же тикет в «Активных»
      - [ ] В шапке статус: *Изучает специалист*
- [ ] Поменять на «Готовится ответ», потом «Закрыто» — проверить оба раза у пользователя

---

## 8. Админ отвечает на тикет

- [ ] В админ-панели открыть тикет
- [ ] Нажать «Ответить пользователю»
- [ ] Написать: *«Это ответ поддержки»*
- [ ] Приходит подтверждение: `📤 Ответ отправлен...`
- [ ] **Проверить у пользователя:**
      - [ ] Пришло сообщение от бота про ответ поддержки
      - [ ] В истории тикета появилась строка: *Служба Поддержки: ...*

---

## 9. «Написать пользователю» — новый диалог от поддержки

- [ ] В админ-панели → «Написать пользователю»
- [ ] Ввести ID тестового пользователя (число)
- [ ] Написать первое сообщение: *«Здравствуйте, это служба поддержки»*
- [ ] Проверить у админа:
      - [ ] Создался тикет с порядковым номером
      - [ ] В шапке: «От кого: Служба Поддержки», «Кому: имя пользователя»
      - [ ] Тикет виден в «Активных»
- [ ] Проверить у пользователя:
      - [ ] Пришло сообщение про открытый тикет от поддержки
      - [ ] Тикет виден в «Активных»
      - [ ] Можно ответить на него

---

## 10. Авто-статус «На рассмотрении» не перезаписывает ручной

- [ ] Взять тикет, где админ **уже сменил статус вручную** (например, «Изучает специалист»)
- [ ] Открыть у пользователя этот тикет → Ответить
- [ ] Написать: *«Ещё вопрос»*
- [ ] Убедиться, что **НЕ** пришло `✅ Тикет №... → На рассмотрении`
- [ ] Статус остался «Изучает специалист» (не сбросился)

---

## 11. Вложения (медиа)

- [ ] Пользователь создаёт тикет с фото/файлом
- [ ] Админ видит 📎 в истории
- [ ] Админ отвечает с фото/файлом
- [ ] Пользователь видит вложение
- [ ] Админ через «Написать пользователю» создаёт тикет с вложением
- [ ] Пользователь видит вложение

## 12. Архивация

- [ ] Админ закрывает тикет (статус «Закрыто»)
- [ ] Тикет исчезает из «Активных» у админа
- [ ] Появляется в «Архив» у админа
- [ ] То же самое у пользователя

---

## 13. Повторный заход — 🟢 обновляется

- [ ] Зайти обычным пользователем в бота
- [ ] Подождать 5+ минут
- [ ] Зайти снова
- [ ] В админ-панели → Пользователи → 🟢 горит у этого пользователя
